계정 및 데이터 삭제 안내 - 넛지Edu

넛지Edu 앱에서는 사용자가 앱 내부에서 계정을 직접 삭제할 수 있는 기능을 제공합니다. 개인정보 보호를 최우선으로 하여, 사용자가 계정 삭제를 요청할 경우 모든 데이터를 신속히 삭제 처리합니다.

1. 앱 내에서 계정 삭제하는 방법

계정을 삭제하려면 다음 단계를 따라주세요:

  1. 앱을 실행합니다.
  2. 하단의 네브바에서 프로필 아이콘을 클릭합니다.
  3. 프로필 하단에 있는 계정 삭제 버튼을 누릅니다.
  4. 계정 삭제 확인 메시지가 표시되면 확인을 누르면 삭제가 완료됩니다.

2. 삭제되는 데이터 유형

계정 삭제 시 아래 데이터를 삭제 처리합니다:

  • 계정과 관련된 모든 정보

3. 문의 사항

계정 삭제와 관련하여 추가 문의 사항이 있으시면 tbot17115@gmail.com으로 언제든지 연락해 주세요.
