계정 및 데이터 삭제 요청 - [앱 이름]

여기에서 사용자는 계정 및 관련 데이터를 삭제 요청할 수 있습니다.

1. 계정 삭제 요청 절차

계정을 삭제하려면 아래 단계를 따라주세요:

  1. 계정 이메일 주소와 함께 삭제 요청을 보내주세요.
  2. 요청을 확인한 후, 7일 이내에 계정을 삭제하고 확인 이메일을 보내드립니다.

2. 삭제/보관되는 데이터 유형

삭제되는 데이터 유형:

  • 이름
  • 이메일 주소
  • 사용 기록

보관되는 데이터:

  • 법적 요구사항에 따라 보관되는 데이터 (최대 1년)

자세한 문의 사항이 있으시면 [연락처 정보]로 연락해주세요.
